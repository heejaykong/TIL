# Javascript 'change' event

HTMLElement 중 `<input>`, `<select>`, and `<textarea>` elements 에 사용할 수 있는 change event를 알게 됐다. 사용자가 해당 element의 value를 바꾸는(change하는) 족족 handler 함수를 실행할 수 있다.
